# 🔐 Login Authentication System

## 📌 Project Overview

This project is a **Login Authentication System** developed using **HTML, CSS, and JavaScript** as part of the **Oasis Infobyte (OIBSIP) Web Development Internship – Level 2 Task 4**.

The application provides a simple authentication mechanism that allows users to register, log in, and access a secured dashboard. It demonstrates the implementation of user authentication concepts, form validation, session management, and protected page access.

---

## 🚀 Features

* User Registration
* User Login
* Logout Functionality
* Form Validation
* Secure Dashboard Access
* Session Management
* Responsive User Interface
* Clean and Modern Design

---

## 🛠️ Technologies Used
